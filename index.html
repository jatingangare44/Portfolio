Projects

  • Lost n’ Found App

    An Android app designed to help users find and claim lost items in my village. Built using Firebase Authentication, Realtime Database, and Cloud Storage, with separate roles for users and authenticators. The app allows finders to submit lost items to assigned authenticators who verify and list them. Claimants can search and request items, and once verified, the item's status is updated in the database. The app ensures transparency and helps reduce unclaimed lost property through a streamlined digital workflow.

  • Fitness Tracker (Java EE)

    A web-based fitness tracking system developed using Java EE, JSP, and MySQL. Users can log workouts, track calories, and monitor progress through a dynamic and responsive interface. The platform supports personalized user accounts, goal setting, and performance analytics. It ensures secure data handling and offers an intuitive dashboard for health insights.

  • RF Signal Trans-Receiver

    Created a wireless communication system using Arduino and RF modules for sending sensor data across devices. Used in various IoT and automation experiments to demonstrate real-time data transmission. The system includes a transmitter that collects data from sensors and a receiver that processes and responds to the signal, such as triggering LEDs or relays. It showcases fundamental concepts of RF pairing, encoding/decoding, and noise handling in wireless systems.
